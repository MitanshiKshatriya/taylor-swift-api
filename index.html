TAYLOR SWIFT API

Open Source API for Taylor Swift lyrics

API End-Points

  • GET RANDOM LYRICS: get
    • QUERY PARAMETERS:
    • GET RANDOM LYRICS FROM AN ALBUM: get?album=<album>
    • GET RANDOM LYRICS FROM A SONG: get?song=<song>
  • GET ALL LYRICS MATCHING THE FILTERS:
    • FILTER BY SONG: get-all?song=<song>
    • FILTER BY ALBUM: get-all?album=<album>
